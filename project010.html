实验室介绍 Laboratory Introduction

之江实验室： 之江实验室（Zhejiang Lab），成立于2017年9月，坐落于杭州城西科创大走廊核心地带，是由浙江省人民政府主导举办、浙江大学等院校支撑、企业参与的事业单位性质的新型研发机构，是浙江深入实施创新驱动发展战略、探索新型举国体制浙江路径的重大科技创新平台 。2020年7月，之江实验室获批牵头建设智能科学与技术浙江省实验室。2021年6月，之江实验室纳入国家实验室体系 。之江实验室正式成为浦江（人工智能）国家实验室基地 之一。之江实验室以“打造国家战略科技力量”为使命，主攻智能感知、人工智能、智能计算、智能网络和智能系统五大科研方向，重点开展前沿基础研究、关键技术攻关和重大装备系统研发，致力于建设国际一流的智能感知研究与实验中心、国际一流的人工智能创新中心、国际一流的智能科学与技术研究中心和全球领先的智能计算基础研究与创新高地 。

项目背景 Program Background

实现“碳达峰、碳中和”战略性目标已被列入国家的“十四五”规划。绿色清洁能源得到大规模开发和利用，储能系统可平抑电网波动，提高新型能源电力系统可靠性。电子产品不可避免的需要在各种环境下使用，外力、温度和湿度等的波动都会对电池的安全造成影响。二次储能电池是大型储能电站与新能源汽车的重要组成部分，但由于安全问题严重限制了其实际应用和发展。     

项目介绍 Program Description

低温锂电池是一种采用特殊材料及工艺制成的，因具有质量轻,比能量高及寿命长等优点得到广泛应用,低温锂电池是一种采用特殊材料及工艺制成的，适合于零下的寒冷环境使用，常用于装备部队、航空、航天、深海潜航设备电源、极地科考、探险设备供电、寒带抢险、救灾用电源、防寒服、防寒鞋电源灯。在本项目中，同学们将在导师的带领下，一步一步深入探索纤维型锂电池和钠电池电极材料制备，帮助同学们深入了解电池电极材料的研究技术和方法，探索其研发和应用的流程。

项目大纲 Syllabus

1.储能电池分类及原理 ：系统讲解储能电池研究前沿，介绍电池发展中面临的安全问题，以及相关基本原理。"

2.储能材料制备所使用的设备简介：一一讲解实验设备，包括功能和原理。设备包括管式炉、马弗炉等、气瓶等。"

3.提高电池安全的解决方案及相关理论分析：分类介绍电池，金属、锂离子、钠离子、锂硫等。归纳总结电池安全共性问题。"

4.如何装配电池：如何制备电极材料，涂覆电极，组装电池。设备包括手套箱、涂覆器、真空干燥箱等。"

5.电池性能测试：电池性能测试所使用的设备介绍,及测试参数设置。相关设备有Neware和蓝电电池性能测试系统和VMP3电化学工作站。参数设置包括电流、电压等。"

6.电池装配：制备电极材料。培养理论到实践的能力，从实验方案设计到完成材料合成。包括文献查阅、背景调研、实验技术路线设计、耗材和原材料准备等。通过电池组装实践，学习电池内部结构，实际操作电池组装，提高动手能力。"

7.电池测试：对组装的电池进行测试，包括阻抗、CV、倍率和循环的数据。"

8.学习如何使用科研软件绘图：学习科研软件使用，包括ORIGIN,Photoshop等"

9.讲解如何分析测试数据：对电池测试结果进行分析，包括阻抗、CV、倍率和循环等的分析方法，绘制图片。
