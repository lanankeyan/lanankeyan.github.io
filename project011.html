项目背景 Program Background

树脂渗透陶瓷复合材料是一种新型新型高性能齿科修复材料，与原生牙齿更为接近，由于材料微观结构以及制备工艺与传统树脂基复合材料有较大差异，因而造成了两者在性能与临床上表现的不同。临床上可以应用于制作单个牙齿的永久性美学修复体，例如贴面、嵌体及高嵌体、前后牙单冠和种植体上部结构等多种修复类型。 除力学性能之外，抗菌性能是牙科材料的又一评价指标。目前在临床实践中使用的牙科树脂渗透陶瓷基质本身并不具有抗菌性。植入的修复材料（如表面粗糙的陶瓷）在术后容易受到细菌感染，从而引起龋齿，进而导致相关的牙周病，严重影响人们的口腔健康和生活质量。研究表明，透明质酸对口腔中的细菌有抑制作用，透明质酸改性的齿科修复材料有较好的应用前景。

项目介绍 Program Description

本项目主要研究新型齿科与骨科修复材料，目前市场上尚无符合原生牙齿力学的修复材料先例，需要设计具有仿生力学梯度的复合材料成分与结构并实现制备，并深入了解材料成分构成对其力学增韧和断裂失效的影响机制，从而获得复合材料性能优化提升的一般准则。同时，研发该种复合材料规模化连续生产的关键工艺、实现具有高稳定性的齿科修复材料的宏量制备。当前，高性能齿科修复材料被欧美等发达国家垄断，我国在原材料的合成、复合材料的设计、高端制备工艺、稳定性控制方面受制于人，急需从原材料到产品全链条实现“国产替代”，以解决在齿科材料领域的“卡脖子”问题。通过引入对口腔微生态起重要作用的透明质酸，赋予口腔修复材料以抗菌性能。

项目大纲 Syllabus

1）高性能树脂渗透陶瓷齿科修复材料研究，透过不同的材料体系与工业方案，制备力学性能、生物性能及美学性能优异的齿科修复复合材料；

2）尽管一些商用齿科材料已实现了较为优异的力学性能，但具有仿生梯度的齿科修复材料尚不成熟，精细控制复合材料梯度的方案仍然缺乏。因此，需要对部分烧结的氧化锆多孔陶瓷胚体内部形貌和聚合物渗透过程的精细控制，实现复合材料在上述力学性能区间的连续调控，并在此基础上对复合材料的聚合物/陶瓷成分、含量、复合相形态和网络结构的空间分布进行设计，开发可控的梯度复合材料制备新工艺，获得符合天然牙齿生物学梯度的修复材料；

3）聚合物/陶瓷复合的齿科修复材料核心指标为力学性能，其受复合材料原材料的本征性能、原材料的比例与分布、树脂渗透技术、复合形态与网络结构、复合界面质量等多方面因素影响，而当前缺乏这些变量与复合材料力学性能和失效机制的全面理解。一方面，通过多相复合材料的理论计算模拟高通量地覆盖变量空间，获得材料成分结构和力学性能的关联；另一方面，通过实验制备验证上述规律，对代表性复合材料进行服役试验，分析包括拉伸、压缩、弯曲和表面硬度，表征弹塑性形变和失效断裂的时空过程，理解其增韧和失效机理，形成一般规律。这些机理性研究将为树脂渗透陶瓷在口腔中的长期使用提供理论支撑，为开发下一代性能更为优异的树脂渗透陶瓷产品提供指导；

4）基于无机金属离子、透明质酸、天然高分子和压电材料的抗菌特性，从力学性能优异和高效抗菌角度设计构建硅酸铝钠基树脂渗透复合陶瓷材料体系，研究复合陶瓷材料体系中抗菌成分（氧化锌/氧化铜/铌酸钾钠/透明质酸等）含量和分布、陶瓷骨架结构和树脂渗透工艺对强韧瓷材料抗菌性能、力学性能等调控规律，明确影响材料力学性能和抗菌性能的关键特征参数；

5）新型齿科骨科修复材料的产业化；

6）项目答辩与点评：学生项目汇报与答辩、导师点评与指导
