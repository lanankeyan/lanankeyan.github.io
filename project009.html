项目背景 Program Background

肿瘤（Tumor）是机体在各种致癌因素作用下，局部组织的某一个细胞在基因水平上失去对其生长的正常调控，导致其克隆性异常增生而形成的新生物。一般认为，肿瘤细胞是单克隆性的，即一个肿瘤中的所有瘤细胞均是一个突变的细胞的后代。 一般将肿瘤分为良性和恶性两大类。良性肿瘤一般称为“瘤”，恶性肿瘤来自上皮组织者称为“癌”，来自间叶组织者称为“肉瘤”。某此些恶性肿瘤也可称“瘤”或“病”，如恶性淋巴瘤、精原细胞瘤、白血病、何杰金氏病等。所有恶性肿瘤习惯称为癌症或癌肿（cancer）。 肿瘤的作用机制非常复杂，涉及到基因突变、细胞凋亡、炎症反应、血管生成等多个方面。 1.基因突变和表达异常：肿瘤细胞中的基因突变和表达异常是肿瘤形成的主要原因之一。突变和异常表达的基因包括肿瘤抑制基因和肿瘤促进基因等，它们的突变和异常表达会导致细胞增殖和分化失控，从而引起肿瘤的形成和发展。 2.细胞凋亡异常：细胞凋亡是细胞自我调节的一种重要机制，对于维持组织和器官的正常结构和功能具有重要作用。肿瘤细胞的凋亡异常是肿瘤形成的另一个重要原因，它会导致肿瘤细胞的无限增殖和分化失控。 3.炎症反应：炎症反应是机体对于内外部刺激的一种自我保护机制，但是长期的炎症反应会导致细胞的损伤和死亡，从而增加肿瘤的发生风险。 4.血管生成异常：血管生成是肿瘤生长和转移的重要机制之一。肿瘤细胞会通过分泌血管生成因子等物质促进血管生成，从而为肿瘤提供充足的营养和氧气，促进肿瘤的生长和转移。 对于肿瘤分子机制的深入研究，可以为肿瘤的预防、诊断和治疗提供更有效的方法和手段。

项目介绍 Program Description

通过本项目，学生不仅可以学习到肿瘤的分子机制，药物作用，科研设计和实验方法，同时可以通过科研项目锻炼提高学生的科学研究能力。本项目可有效提升学生逻辑思辨能力，生物化学与分子生物学、细胞生物学的知识储备，分析问题及解决问题的能力等，同时也需要学生具备一定的自学和钻研能力，这将为该学生的科研和技术工作能力带来极大的提高。

项目大纲 Syllabus

任务一：生命科学导论与文献阅读

根据学生基础，阅读导师布置的学习资料，阅读并整理国内外文献和经典教材的部分章节，了解与掌握了解细胞的结构、功能和生命周期等基础知识，理解肿瘤细胞的形成和发展机制。

了解DNA、RNA、蛋白质等分子的结构、功能和调控机制，理解肿瘤基因的突变和表达异常等分子机制。

任务二：肿瘤靶点与分子机制-深度学习

在掌握了基础知识后，学生将会在该领域具有一定的基础，此时将进入深度学习的学习环节。学生将阅读导师布置的学习资料，阅读并整理国内外文献和经典教材的部分章节，了解与掌握肿瘤影响因素、发生发展机制、诊断和治疗方法（肿瘤标志物检测、影像学检查、手术切除、放疗、化疗、靶向治疗、免疫治疗）。了解肿瘤细胞内的一些分子靶点和信号通路，可以通过针对这些靶点的药物干预来抑制肿瘤细胞的生长和扩散

任务三：实验技术的学习与应用

细胞实验技术：细胞培养（细胞复苏、冻存、细胞培养）细胞凋亡检测、wound healing实验、transwell实验。

免疫实验技术：western bloting实验，免疫荧光实验、免疫组化实验、ELISA实验

基因工程技术：基因载体构建、PCR、RNA干扰等实验技术。

任务四：博士论文精读

讲解博士论文让学生了解如何确定科研思路和科研方向，科研需要哪些准备，如何做文献调研，摘要和引言部分如何撰写。实验材料和方法的选用，实验结果的分析讨论，如何在现有的基础上进一步深入研究。

任务五：项目答辩与点评

学生项目汇报与答辩、导师点评与指导
