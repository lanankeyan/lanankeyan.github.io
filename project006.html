项目背景 Program Background

随着全球气候持续变暖，过去几十年生产化石能源和应用对环境负面的影响不断加速，化石燃料的大量消耗以及由此产生严重的环境污染问题日益引起人们的关注，在实现可持续发展的前提下，为满足人类社会日益增加的能源需求，发展可再生能源及其转化技术势在必行。氢气凭借其高能量密度、无污染等优势，应用领域和场景十分丰富，在碳达峰碳中和目标的背景下，氢能的多场景大规模应用将助力降碳目标早日实现。

项目介绍 Program Description

本项目首先讨论能源与环境和可持续发展的关系，不可再生能源对环境的影响，认识管理好地球资源的重要性和其他可持续发展的重点领域。探讨如何开发氢能源以达到2060年中国零碳排放的目标。深入研究氢能催化材料在航空航天，电力，交通等领域的应用新范式。

项目大纲 Syllabus

1）能源与环境和可持续发展的关系：认识目前不可持续发展的状况、慎重管理好地球的资源的重要性、不可再生能源对环境的影响、可持续发展的要素；

2）不可再生与可再生能源生产过程、不可再生能源生产及应用对环境的影响、目前如何满足中国发展对能源的需求、中国零碳排放的远景和路线图、开发氢能对可持续发展的必须性；

3）中国氢能的现状和未来的发展、电解水器件的优点和缺点；

4）如何结合催化材料材料研究、深入探讨如何建立氢能源催化材料在航空航天，电力，交通等领域的应用新范式。

5）项目答辩与点评：学生项目汇报与答辩、导师点评与指导
