项目背景 Program Background

近年来，随着我国“碳达峰”“碳中和”战略的提出，“低碳经济”“绿色金融”，等成为了未来社会发展的必然趋势。园区作为密集碳排放生产活动的承担主体，其绿色低碳发展也自然成为了实现“双碳”目标的重要途径。而随着全球碳交易市场机制的不断完善，碳排放权已经成为一种商品，与有形商品一样由供求关系决定价格，因此具有了价值与金融属性，成为了企业的碳资产，从而有可能为园区带来预期的经济利益或形成园区的现时义务。因此，落实园区碳中和建设与企业绿色金融开发，是企业践行ESG理念，实现绿色可持续发展的必经之路。

项目介绍 Program Description

本项目围绕山东某医疗器械龙头企业园区的低碳改造与提升（规划研究面积8453亩、既有建筑面积120万㎡）进行研究。首先对企业既有的医疗产业链进行分析与延申，提出大健康产业的绿色转型策略。其次，在产业绿色发展引领下，研究园区新建部分的功能业态落位，交通组织优化与新建碳中和展示馆设计。在功能落位前提下，开展既有园区的绿化改造与提升，如优化用能，提升清洁能源使用率，并分别在建筑、交通、循环、生态、能源、负碳技术等领域开展低碳转型研究。最后，对企业进行全方位碳排放盘查，核算园区现状碳排放并制定碳达峰与碳中和目标时间点与行动方案，力争在国家双碳目标之前实现既定目标。为更好发挥企业绿碳经济效益，对企业进行碳资产管理，并围绕绿色金融发展趋势研究如何将碳资产金融化与交易。

项目大纲 Syllabus

1）企业健康产业现状、大健康产业链内涵与价值影响、大健康产业链绿色转型的强链、固链、补链策略、大健康产业链绿色发展建设运营的实施方案；

2）国家碳达峰试点园区建设方案研究、企业建设碳达峰园区的绿色基础、企业碳达峰园区碳中和目标与路径搭建；

3）绿色产业导入研究、建筑现状耗能分析与绿色建筑建设策略；可再生能源利用现状与优化用能策略研究、循环水处理与固废处理低碳化策略、园区交通绿色发展策略与行动方案研究、绿色生态建设与绿色创新技术应用策略研究；

4）国家高标准碳达峰示范园区空间规划设计、国家碳中和展览馆设计，碳中和双控数字化智慧平台搭建体系研究；

5）企业碳资产盘查方式研究、企业碳排放核算研究、企业减碳核算研究、企业碳资产识别与筛选、碳资产开发与管理、企业碳资产交易策略研究、成本与风险管控指导；

6）项目答辩与点评：学生项目汇报与答辩、导师点评与指导
