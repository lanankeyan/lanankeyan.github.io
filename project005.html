项目背景 Program Background

随着全球气候持续变暖，过去几十年生产化石能源和应用对环境负面的影响不断加速，虽然气候变化与全球暖化已经成为多个国家热门论题，但还是缺乏足够影响力和切实的行动。中国为发展中的大国以2060为目标设立了能源与环境和可持续发展路线图。

项目介绍 Program Description

本项目首先讨论能源与环境和可持续发展的关系，不可再生能源对环境的影响，认识管理好地球资源的重要性和其他可持续发展的重点领域。探讨如何开发低碳能源以达到2060年中国零排放的目标。深入研究反铁电储能材料在航空航天，电子元器件，自动驾驶等领域的应用新范式。

项目大纲 Syllabus

1）能源与环境和可持续发展的关系：认识目前不可持续发展的状况、慎重管理好地球的资源的重要性、不可再生能源对环境的影响、可持续发展的要素；

2）不可再生与可再生能源生产过程、不可再生能源生产及应用对环境的影响、目前如何满足中国发展对能源的需求、中国零排放的远景和路线图、开发反铁电对可持续发展的必须性；

3）中国反铁电的现状和未来的发展、反铁电器件的优点和缺点；

4）如何结合反铁电材料研究、深入探讨如何建立反铁电储能材料在航空航天，电子元器件，自动驾驶等领域的应用新范式。

5）项目答辩与点评：学生项目汇报与答辩、导师点评与指导
