项目背景 Program Background

科学是知识体系，技术是以知识为基础的手段，而工程则是有目的、有组织地改造世界的活动。零碳建筑是由世界绿色建筑理事会提出的“高能效建筑，完全由现场或场外可再生能源供电。”零碳建筑考虑的不仅仅是建筑运行阶段的碳排放，更是全面考虑建筑建造过程中的隐含碳排放，目标是在建筑的全生命周期中实现碳的零排放。

项目介绍 Program Description

绿色建筑是土木工程的热门研究方向，着重提高资源的利用效率，采用创新、健康的设计方法，基于生态学为依据的建筑，同时注重建筑的消防安全设计，以及工程机器人的设计和应用，还涉及工程管理的研究，其跨越项目的整个生命周期，需要对管理、经济学、计算机科学和工程学的定性和定量原则有深刻的理解。该课程中也有一些设计和编程作业，以及小组项目，帮助学生应用所学知识解决实际问题，锻炼团队合作能力。
