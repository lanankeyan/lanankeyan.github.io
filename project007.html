项目背景 Program Background

液流电池是一种新的蓄电池，是利用正负极电解液分开，各自循环的一种高性能蓄电池。具有容量高、使用领域(环境)广、不自燃和循环使用寿命长的特点。是目前的一种新能源产品。液流电池是一种正在积极研制开发的新型大容量电化学储能装置，它不同于通常使用固体材料电极或气体电极的电池，其活性物质是流动的电解质溶液，它最显著特点是规模化长时蓄电（≥4h）。

项目介绍 Program Description

本项目围绕国创新能源研究院下属单位嘉兴纳科新材料有限公司的液流电池电极材料项目，对现有产品做更进一步的优化。

项目大纲 Syllabus

1）液流电池电极毡的活化研究与大生产工艺研究；

2）液流电池电极布的活化研究与大生产工艺研究；

3）液流电池双极板的制备工艺研究与大生产工艺研究；

4）液流电池电极材料标准化研究；

5）液流电池电极材料的未来发展趋势研究。
